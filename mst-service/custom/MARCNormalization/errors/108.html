The service produced no output for this particular record. This may be due to some problem with the structure of the record’s XML.



When a service produces no output for a specific input record, this error message is attached to the input record. To view all of the records from a particular service or repository that produced no output in a subsequent service, select the original service or repository as a facet value before viewing records that have the error. For example, to see records that received no output from MARC Transformation, select the facet value for MARC Normalization. To see records that received no output from MARC Normalization, select the facet value for the original harvested repository.
